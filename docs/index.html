setup · about five minutes

Get it running

Copy the board. Open the template sheet and choose File ▸ Make a copy.
Open the script editor. In your copy: Extensions ▸ Apps Script. Delete the sample code.
Paste the sync script. Copy CalendarSync.gs from the repo, paste, and save.
Authorize once. Pick syncToCalendar, click Run, approve the calendar permission.
Turn on auto-sync. Triggers ▸ Add Trigger ▸ syncToCalendar ▸ Time-driven ▸ every 5 minutes.

good to know

How to live with it

Add and edit tasks on the Tasks sheet only — leave the Synced? column blank; the script stamps it for you. The Board View is a read-only mirror. To clear finished tasks, tick Done rather than deleting rows, so each event stays matched to its task.
